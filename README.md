# ReefScan Landing Page

A modern, responsive landing page for the ReefScan iOS app — an AI-powered scanner for reef aquariums.

## Overview

ReefScan helps reef enthusiasts identify fish, corals, pests, and problems in their aquariums using AI-powered image recognition.

**Live App:** [App Store](https://apps.apple.com/app/reefscan)

## Tech Stack

- **React 18** — UI framework
- **TypeScript** — Type safety
- **Vite** — Build tool & dev server
- **Tailwind CSS** — Utility-first styling
- **Framer Motion** — Animations

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
